📚 学习 Git 和 GitHub

Git 是一个分布式版本控制系统，用于跟踪文件更改，尤其适合开发协作项目。GitHub 则是一个基于 Git 的代码托管平台。

🔧 安装 Git

  • Windows: 下载 Git for Windows
  • macOS: 使用 Homebrew 安装：brew install git
  • Linux: 使用包管理器，例如 sudo apt install git

🚀 常用 Git 命令

  • git init：初始化本地仓库
  • git clone [url]：克隆远程仓库
  • git add .：添加所有更改
  • git commit -m "提交信息"：提交更改
  • git push：推送到远程仓库
  • git pull：拉取最新更改

🌐 创建 GitHub 仓库

  1. 注册并登录 GitHub
  2. 点击右上角 + 图标，选择 New repository
  3. 填写仓库名称、说明，并选择是否公开
  4. 创建后可使用 git remote add origin [url] 绑定远程仓库
